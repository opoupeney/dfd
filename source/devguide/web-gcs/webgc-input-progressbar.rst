Progress Bar
============

.. image:: ../../images/icons/input_progressbar.png
    :width: 50px
    :height: 50px


A progressbar control is used to show progress, where the user is in a series of steps, a task or a process, such as a
download or file transfer. Progressbar are

Typically, progress bars are not clickable.
