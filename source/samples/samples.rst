DreamFace Samples
=================

DreamFace Samples allow you to quicly explore DreamFace's features and components. These samples are provided as a set of useful ready-to-use views and their dependencies. Samples can be individually installed in your DreamFace applicaton:

- From DreamFace Studio's toolbar, click the help icon and select the 
- Browse all samples, click the ``Download`` icon to install the sample in your existing DreamFace app
- A popup window appears, select your application from the list and click ``Install``
